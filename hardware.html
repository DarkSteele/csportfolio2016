CPU (Central Processing Unit) RAM (Random Access Memory) Hard Drive
The Central Processing Unit (CPU) is the part of a computer where operations are controlled, interpreted, and executed. Random Access Memory (RAM) is a type of computer memory that can be accessed at random. Any byte of memory can be accessed without affecting the other bytes. It is the most common type of memory found in computers. Hard Drive is a high-capacity, self-contained storage device used to read hard disks. It contains read-write mechanisms, plus one or more hard drive.
BUS Motherboard Mouse, Keyboard, Monitor
A computer BUS connects all the internal components of a computer to the motherboard. It transfers information and memory, as well as other aspects. The motherboard contains and holds together crucial parts of the computer, such as the CPU and memory. This is one of the most important parts of the computer/device, and controls most actions. The mouse is an input device that moves an on-screen cursor to different items and texts. It is used to select, move, highlight, and manipulate the items on the screen. The keyboard is a typewriter-like device that uses an arrangement of number, letter, and symbol keys to input text and other items onto the computer. Using the keyboard, I was able to make this website! The computer monitor is the display screen used to show visual output from a computer, camera, and other video generating devices.
